# slides

発表に使うスライドをまとめています。

## 非公開スライドへのアクセス

非公開スライドは [sops](https://github.com/getsops/sops) を利用して暗号化されています。
もしアクセスしたい場合には、[age](https://github.com/FiloSottile/age)鍵を該当するスライドのディレクトリに配置されている`.sops.yaml`に追加するPull Requestを送信してください。
